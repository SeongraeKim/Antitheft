도난방지 App
============
2020-04-05 - (~ing)

> ### 설명
> 스마트폰 충전중 타인이 충전기를 분리시켰을 경우!  
> 스마트폰 움직임이 감지되었을 경우!  
> 경고음이 발생하여 도난을 확인하실 수 있습니다.

> ### 구현내용
> + UI 구현
> + 충전기 해제 시 알림 구현

> ### 추가 구현해야 할 사항
> + 움직임 감지기능
> + 볼륨 고정
> + 잠금기능
> + 광고 추가 및 구글플레이 등록
